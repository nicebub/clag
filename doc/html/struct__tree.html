Detailed Description

Structure Definition

The tree _tree structure is used to wrap the tree of _node's with a pointer to the root of the tree _node* root, and a size and depth to the tree. This struct is used very little at this time, and is saved for future use.

struct _tree 

struct _node *root: A pointer to the root _node of the tree

int depth: The depth of the tree, ie, the maximum edges from the,
            lowest child to the root.

int size: The total amount of _nodes in the tree.
